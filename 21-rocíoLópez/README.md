*_1834 - 1837_* MAQUINA ANALITICA (Charles Baggabel) Construida para realizar calculos establecidos anteriormente. Conciderada la primera maquina de calculo automatico. 

*_1801_* Telar De Jaquar (Joseph Marie Jacquard) telar. Primer sistema de automatizacion binaria, es decir, existe un detonante (dar la accion) dentro de el funcionamiento. Para este caso, se utilizaban cartas perforadas que permitian el paso de la aguja dentro dwel el telar. *si hay agujero* La varilla pasa, se activa el gancho y teje el hilo. *si no hay agujero* la varilla choca sin generar cambios en el tejido-
