**historia**

**maquina diferencial** 

*_1822_* MAQUINA DIFERENCIAL (Charles Babbage) Maquina para sistematizar el calculo y la tabulacion (tablas) compuesto por engranajes quese usaban para resolver ecuaciones e imprimirlas como tablas, pero no fue terminada.


*_1991_* MAQUINA DIFERENCIAL II   El museo de ciencias de londres contruyo la maquina usando los planos originales, demostrando su funcionalidad y el gran avance para la epoca victoriana. 

*_1834 - 1837_* MAQUINA ANALITICA (Charles Baggabel) Construida para realizar calculos establecidos anteriormente. Conciderada la primera maquina de calculo automatico. 

*_1801_* Telar De Jaquar (Joseph Marie Jacquard) telar. Primer sistema de automatizacion binaria, es decir, existe un detonante (dar la accion) dentro de el funcionamiento. Para este caso, se utilizaban cartas perforadas que permitian el paso de la aguja dentro dwel el telar. *si hay agujero* La varilla pasa, se activa el gancho y teje el hilo. *si no hay agujero*
